Equações do 1º Grau: O Segredo da Balança

Uma equação do 1º grau é uma igualdade usada para descobrir um valor desconhecido, representado por uma letra (incógnita como o X).

Formato Padrão: ax + b = 0

O objetivo principal é isolar a letra. Pense nela como uma balança: tudo o que você faz de um lado, deve fazer do outro. Tudo que muda de lado da igualdade inverte sua operação (o que está somando passa subtraindo, e o que está multiplicando passa dividindo).

Função do 1º Grau: Linhas Retas e Comportamentos

A função de 1º grau (ou função afim) relaciona duas variáveis e seu gráfico gera sempre uma linha reta.

Fórmula Geral: f(x) = ax + b

O segredo para saber o comportamento do gráfico está no valor de 'a' (coeficiente angular):

  • Função Crescente: Ocorre quando a > 0 (positivo). Conforme o valor de X aumenta, o valor de Y também aumenta. A reta sobe da esquerda para a direita.
  • Função Decrescente: Ocorre quando a < 0 (negativo). Conforme o valor de X aumenta, o valor de Y diminui. A reta desce da esquerda para a direita.

Equações do 2º Grau e a Fórmula de Bhaskara

Identificada quando o maior expoente da incógnita é 2 ($x^2$). A fórmula de Bhaskara é o método mais conhecido para encontrar as raíces (os valores de X que zeram a equação).

Estrutura: ax² + bx + c = 0
Delta: Δ = b² - 4ac

Dependendo do valor de Δ (Delta), a equação pode ter duas soluções reais, uma única solução ou nenhuma solução real.

Função do 2º Grau: Curvas, Parábolas e Vértices

A função de 2º grau (quadrática) possui o formato $f(x) = ax^2 + bx + c$ e seu gráfico é uma curva chamada parábola.

A Concavidade e o Comportamento

  • Se a > 0 (Positivo): A concavidade é voltada para cima (formato de sorriso).
  • Se a < 0 (Negativo): A concavidade é voltada para baixo (formato de montanha).

O Vértice da Parábola

X do Vértice: X_v = -b / 2a
Y do Vértice: Y_v = -Δ / 4a

Sistemas de Equações e o Número de Incógnitas

Quando temos duas variáveis desconhecidas (X e Y), formamos um sistema. Graficamente, a resposta de um sistema linear é exatamente o ponto onde duas retas se cruzam (como na imagem acima).

  • Método da Substituição: Isolamos uma das letras em uma equação e jogamos na outra.
  • Método da Adição: Somamos ou subtraímos as duas equações para anular temporariamente uma variável.
